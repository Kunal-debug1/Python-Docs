# Python Documentation Website

A modern, responsive, and beginner-friendly Python Documentation Website built to help learners and developers explore Python concepts from basics to advanced topics in a clean and structured way.

---

## 🚀 Features

* 📘 Beginner to Advanced Python Notes
* 🎨 Modern UI with Responsive Design
* 🔍 Search Functionality
* 📂 Organized Topic-wise Documentation
* 💡 Code Examples with Syntax Highlighting
* ⚡ Fast and Lightweight
* 📱 Mobile Friendly

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/python-docs-website.git
```

### Navigate to Project

```bash
cd python-docs
```

### Run Website

If using only frontend:

```bash
open index.html
```
